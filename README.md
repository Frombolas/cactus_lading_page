# cactus_lading_page
Repositório referente ao projeto de extensão DivulgaCactus, que possui como objetivo geral:

Desenvolver uma plataforma digital (site institucional) pública, responsiva e de fácil gerenciamento para centralizar e divulgar as ações, projetos e objetivos do núcleo de extensão CACTUS, ampliando sua visibilidade e facilitando a interação com a comunidade.
